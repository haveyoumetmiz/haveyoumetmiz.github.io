Innovative Car Racing Game

Objective:

Created a car racing game controlled by a webcam-detected circular object (like a plate), replacing traditional keyboard inputs.

Overview:

The game merges physical interaction with virtual gameplay, allowing players to steer and accelerate using real-world movements.

Key Features:

  • Object Detection: Uses OpenCV to track a circular object and translate its movements into in-game controls.
  • Realistic Controls: Rotate the object to steer and tilt it for acceleration.
  • Customizable Gameplay: Multiple tracks, car models, and difficulty levels.

Innovative Control System for Racing Game Using Real-Life Object Detection

Description:

I am developing a cutting-edge racing game using Unity, where the car is controlled by detecting and tracking real-life objects through a webcam using OpenCV. Instead of the traditional WASD keys, the game leverages computer vision to detect a circular object (like a plate) and uses its rotation to steer the car in the game. This creates a unique and immersive gameplay experience, merging physical objects with virtual control.
